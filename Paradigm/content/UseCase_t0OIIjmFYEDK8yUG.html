Scenario
Steps
1. El ADMINISTRADOR selecciona la opción “Registrar Personal” en el sistema.
2. El sistema muestra el formulario de registro de usuario.
3. El ADMINISTRADOR ingresa los siguientes datos:
    3.1. Nombre
    3.2. ID
    3.3. Teléfono
    3.4. Correo electrónico
    3.5. Contraseña
    3.6. Rol (Administrador, Almacenista, Personal Operativo)
4. El ADMINISTRADOR confirma el registro.
5. El sistema valida la información ingresada.
6. if todos los campos obligatorios están completos
    6.1. AND el ID no existe
    6.2. AND el correo no existe
    6.3. AND el formato del correo es válido
    6.4. AND la contraseña cumple con las políticas de seguridad) THEN
    6.5. • El sistema guarda el nuevo usuario.
    6.6. • El sistema asigna el rol seleccionado.
    6.7. • El sistema muestra mensaje: "Usuario registrado correctamente".
7. else
    7.1. • El sistema muestra los mensajes de error correspondientes.
    7.2. • El sistema permite corregir la información.
   end if
8. Fin del caso de uso.
