Scenario
Steps
1. El ADMINISTRADOR accede al módulo de reportes en el sistema.
2. El sistema muestra las opciones de reportes disponibles, incluyendo “Reporte de Ventas”.
3. El ADMINISTRADOR selecciona “Reporte de Ventas”.
4. El sistema presenta filtros y criterios para personalizar el reporte, tales como:
    4.1. Rango de fechas
    4.2. Tipo de venta (contado, crédito, etc.)
    4.3. Categoría de producto
    4.4. Vendedor
5. El ADMINISTRADOR ingresa o selecciona los filtros deseados.
6. El ADMINISTRADOR solicita generar el reporte.
7. El sistema valida los criterios ingresados.
8. El sistema consulta la base de datos con los filtros aplicados.
9. El sistema procesa los datos y genera el reporte, que puede incluir:
    9.1. Total de ventas por período
    9.2. Ventas por producto o categoría
    9.3. Ventas por vendedor
    9.4. Comparativos con períodos anteriores
    9.5. Gráficos y tablas resumen
10. El sistema muestra el reporte en pantalla.
11. El ADMINISTRADOR puede exportar el reporte a formatos como PDF, Excel, o imprimirlo.
12. El caso de uso finaliza.
