Scenario
Steps
1. El MECANICO selecciona la opción **“Hoja de Vida de la Moto”**.
2. El sistema muestra un formulario de búsqueda por **Placa**.
3. El MECANICO ingresa la **Placa de la moto** a consultar.
4. El MECANICO selecciona la opción **“Buscar”**.
5. El sistema valida la placa ingresada:
    5.1. if IF la placa existe en el sistema THEN
        5.1.1. • El sistema carga la ficha técnica de la moto.
        5.1.2. • El sistema muestra el historial completo de la moto, incluyendo:
            5.1.2.1. * Servicios realizados
            5.1.2.2. * Repuestos instalados
            5.1.2.3. * Diagnósticos previos
        5.1.3. • El usuario puede filtrar o ordenar el historial por fecha o tipo de servicio.
    5.2. else
        5.2.1. • El sistema muestra mensaje: "Placa no encontrada".
        5.2.2. • Permite ingresar otra placa.
         end if
6. El MECANICO puede cerrar la hoja de vida o realizar otra consulta.
7. while desee consultar otra moto
    7.1. • Ingresar nueva placa.
    7.2. • El sistema actualiza la información mostrada.
   end while
8. Fin del caso de uso.
