Description

Scenario
Steps
1. El ADMINISTRADOR o ALMACENISTA autorizado selecciona la opción “Registrar Producto”.
2. El sistema muestra el formulario de registro de producto.
3. El ALMACENISTA ingresa la siguiente información:
    3.1. Código / SKU
    3.2. Nombre del producto
    3.3. Marca
    3.4. Categoría
    3.5. Precio de Venta Sugerido
    3.6. Punto de Reorden (Stock mínimo)
4. El ALMACENISTA confirma el registro.
5. El sistema valida la información ingresada.
6. if todos los campos obligatorios están completos
    6.1. AND el Código/SKU no existe en el sistema
    6.2. AND el Precio de Venta Sugerido > 0
    6.3. AND el Punto de Reorden ≥ 0) THEN
    6.4. • El sistema guarda el nuevo producto en el catálogo.
    6.5. • El sistema muestra mensaje: "Producto registrado correctamente".
7. else
    7.1. • El sistema muestra los mensajes de error correspondientes.
    7.2. • Permite corregir la información.
   end if
8. Fin del caso de uso.
