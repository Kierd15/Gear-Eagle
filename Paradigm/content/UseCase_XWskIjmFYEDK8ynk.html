Scenario
Steps
1. El MECANICO) selecciona la opción **“Asignar Mano de Obra”** dentro de una **Orden de Servicio**.
2. El sistema muestra la orden de servicio y la sección de servicios disponibles o previamente agregados.
3. El usuario agrega un nuevo servicio realizado, ingresando la siguiente información:
    3.1. * Descripción del servicio (Ejemplo: cambio de aceite, reparación de frenos)
    3.2. * Tiempo estimado o trabajado
    3.3. * Costo asociado al servicio
4. El usuario confirma la asignación de la mano de obra.
5. El sistema valida la información ingresada:
6. if todos los campos obligatorios están completos
    6.1. AND el tiempo y costo son mayores a cero) THEN
        6.1.1. • El sistema guarda la información de la mano de obra en la orden de servicio.
        6.1.2. • El sistema actualiza el costo total de la orden.
        6.1.3. • Muestra mensaje: "Mano de obra asignada correctamente".
7. else
    7.1. • El sistema muestra mensajes de error correspondientes (Ejemplo: tiempo o costo inválido).
    7.2. • Permite corregir la información.
   end if
8. 6. Fin del caso de uso.
