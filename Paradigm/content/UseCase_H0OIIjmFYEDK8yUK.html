Scenario
Steps
1. ElADMINISTRADOR selecciona la opción “Consultar Usuarios” en el sistema.
2. El sistema muestra el listado completo de empleados registrados.
3. El sistema presenta opciones de filtrado por:
    3.1. Rol (Administrador, Almacenista, Personal Operativo)
    3.2. Estado de disponibilidad (Disponible, No disponible).
4. El ADMINISTRADOR selecciona uno o más criterios de filtro.
5. El ADMINISTRADOR confirma la búsqueda.
6. El sistema procesa los filtros seleccionados.
7. if IF (existen usuarios que cumplen los criterios) THEN
    7.1. • Mostrar listado de usuarios filtrados.
8. else
    8.1. • Mostrar mensaje: "No se encontraron usuarios con los criterios seleccionados".
   end if
9. El ADMINISTRADOR puede modificar o limpiar los filtros.
10. while desee realizar nuevas consultas
    10.1. • Seleccionar nuevos criterios.
    10.2. • El sistema actualiza el listado.
    end while
11. Fin del caso de uso.
