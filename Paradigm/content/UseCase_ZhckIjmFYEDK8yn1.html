Scenario
Steps
1. El MECANICO selecciona la opción **“Solicitar Repuestos”** dentro de una **Orden de Servicio**.
2. El sistema muestra el listado de productos disponibles del **Módulo de Productos**, con su información principal:
    2.1. * Código / SKU
    2.2. * Nombre
    2.3. * Marca
    2.4. * Stock disponible
3. 3. El cliente selecciona uno o más repuestos que necesita para la orden.
4. 4. El cliente indica la cantidad requerida de cada repuesto.
5. 5. El cliente confirma la solicitud de repuestos.
6. 6. El sistema valida la información ingresada:
    6.1. if los repuestos seleccionados existen
        6.1.1. AND la cantidad solicitada ≤ stock disponible) THEN
        6.1.2. • El sistema agrega los repuestos a la orden de servicio.
        6.1.3. • El sistema actualiza temporalmente la cantidad disponible del inventario (reserva de stock).
        6.1.4. • Muestra mensaje: "Repuestos agregados correctamente a la orden".
    6.2. else
        6.2.1. • El sistema muestra mensaje de error: "Cantidad solicitada supera el stock disponible" o "Repuesto no encontrado".
        6.2.2. • Permite corregir la solicitud.
         end if
7. 7. Fin del caso de uso.
