Scenario
Steps
1. El ALMACENISTA accede al módulo Inventario.
2. El sistema muestra las opciones disponibles.
3. El ALMACENISTA selecciona la opción “Modificar Ingreso”.
4. El sistema muestra un formulario de búsqueda con filtros:
    4.1. Número de ingreso
    4.2. Proveedor
    4.3. Fecha
    4.4. Producto
5. El ALMACENISTA ingresa los criterios de búsqueda.
6. El ALMACENISTA presiona Buscar.
7. El sistema muestra el listado de ingresos que coinciden con los criterios.
8. EALMACENISTA selecciona el ingreso que desea modificar.
9. El sistema muestra el detalle del ingreso:
    9.1. Productos
    9.2. Cantidades
    9.3. Precio unitario
    9.4. Lote (si aplica)
    9.5. Fecha
10. El ALMACENISTA realiza las modificaciones necesarias (cantidad, precio, proveedor, etc.).
11. El ALMACENISTAr confirma la modificación.
12. El sistema valida la información ingresada.
13. El sistema actualiza los datos del ingreso en la base de datos.
14. El sistema recalcula:
    14.1. Costo promedio del producto
    14.2. Stock disponible
    14.3. Impacto en costo de venta
15. El sistema registra la modificación en el historial (usuario, fecha, cambios realizados).
16. El sistema muestra mensaje de confirmación.
17. El caso de uso finaliza.
