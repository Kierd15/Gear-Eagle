Scenario
Steps
1. El CLIENTE accede al módulo “Compras”.
2. El sistema muestra el menú de opciones disponibles.
3. ElCLIENTE selecciona la opción “Consultar compras”.
4. El sistema despliega un formulario con filtros disponibles:
    4.1. Rango de fechas
    4.2. Proveedor
    4.3. Producto
    4.4. Categoría
5. Estado de compra
6. El CLIENTE ingresa o selecciona los criterios de búsqueda.
7. ECLIENTE confirma la consulta.
8. El sistema valida los criterios ingresados.
9. El sistema consulta la base de datos.
10. El sistema muestra el listado de compras que cumplen con los criterios, incluyendo:
    10.1. Fecha de compra
    10.2. Proveedor
    10.3. Productos adquiridos
    10.4. Cantidad
    10.5. Precio unitario
    10.6. Total de compra
11. El sistema permite visualizar gráficos comparativos de evolución de precios.
12. ElCLIENTE puede exportar la información (PDF, Excel).El caso de uso finaliza.
