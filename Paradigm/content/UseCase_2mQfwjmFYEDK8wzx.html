Scenario
Steps
1. El usuarioADMINISTRADOR o ALMACENISTA autorizado selecciona la opción “Desactivar Producto”.
2. El sistema muestra el listado de productos activos en el catálogo.
3. El ALMACENISTA Autorizado o El ADMINISTRADOR selecciona el producto o productos que desea deshabilitar.
4. El sistema muestra la información del producto seleccionado (Código/SKU, Nombre, Marca, Categoría, Estado).
5. El ALMACENISTA Autorizado o ElADMINISTRADOR confirma la desactivación del producto.
6. El sistema realiza la desactivación lógica:
    6.1. if el producto está activo THEN
        6.1.1. • Cambiar el estado del producto a “Inactivo”.
        6.1.2. • Impedir que el producto sea seleccionado en nuevas órdenes.
        6.1.3. • Mantener el registro histórico de ventas y órdenes anteriores.
        6.1.4. • Mostrar mensaje: "Producto desactivado correctamente".
    6.2. else
        6.2.1. • Mostrar mensaje: "El producto ya se encuentra desactivado".
         end if
7. Fin del caso de uso.
