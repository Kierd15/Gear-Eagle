Scenario
Steps
1. El ADMINISTRADOR o MECANICO autorizado selecciona la opción **“Registrar Moto”**.
2. El sistema muestra el formulario de registro de ficha técnica de la moto.
3. El MECANICO ingresa la siguiente información:
    3.1. Placa (Obligatorio y único)
    3.2. Marca
    3.3. Modelo / Año
    3.4. Color
    3.5. Número de Motor
    3.6. Kilometraje actual
4. 4. El MECANICO confirma el registro.
5. 5. El sistema valida la información ingresada:
    5.1. if IF la placa es única y no existe en el sistema
        5.1.1. AND los campos obligatorios están completos
        5.1.2. AND el kilómetro actual ≥ 0 THEN
        5.1.3. • El sistema guarda la ficha técnica de la moto.
        5.1.4. • El sistema muestra mensaje: "Moto registrada correctamente".
    5.2. else
        5.2.1. • El sistema muestra los mensajes de error correspondientes.
        5.2.2. • Permite corregir la información.
         end if
6. 6. Fin del caso de uso.
